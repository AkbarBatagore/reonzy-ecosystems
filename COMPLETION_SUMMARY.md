## 🚀 How to Use

### Step 1: Open in Browser
```bash
# Start local server
python -m http.server 8000
# Then open: http://localhost:8000
```

### Step 2: Add Slideshow Images
1. Prepare 12 PNG images (recommended: 1280x400px)
2. Place in `/img/slideshow/`
3. Name: `slideshow-1.png`, `slideshow-2.png`, ..., `slideshow-12.png`
4. Refresh browser

### Step 3: Add App Icons
1. Prepare PNG icons (recommended: 60x60px)
2. Place in `/img/icon-apps/`
3. Name: `apps-1.png`, `apps-2.png`, ..., `apps-N.png`

### Step 4: Add Applications
1. Open `/data/apps.json`
2. Replace sample apps with real applications
3. Update icon references to match your files
4. Refresh browser
